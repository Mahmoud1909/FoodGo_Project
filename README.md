# FoodGo

Complete Food Delivery Management System built with Laravel and Firebase.

## Features

- Admin Panel
- Restaurant Panel  
- Website Panel
- Landing Page
- Firebase Integration
- Real-time Updates

## Requirements

- PHP >= 8.1
- Composer
- MySQL
- Firebase Project

## Installation

```bash
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
```
